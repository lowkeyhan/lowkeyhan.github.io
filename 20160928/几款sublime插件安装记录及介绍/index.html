几款sublime插件安装记录及介绍

　　最近才接触到sublime text，听说过sublime很好用不过需要自己安装很多插件，我又是一个比较懒的人所以就一直没有勇气去试试。
　　最近正好有时间于是准备安装一下看看，这篇文字主要记录了安装了那些插件，算是一个插件列表的备忘吧。担心哪天重新安装的时候还需要重新找插件。
　　废话到这里，开始上插件：

1.SublimeCodeIntel

SublimeCodeIntel - 一个全功能的 Sublime Text 代码自动完成引擎实现代码智能提示及自动完成。支持的语言有：JavaScript, Mason, XBL, XUL, RHTML, SCSS, Python, HTML, Ruby, Python3, XML, Sass, XSLT, Django, HTML5, Perl, CSS, Twig, Less, Smarty, Node.js, Tcl, TemplateToolkit, PHP.

　　安装完sublime后发现自带的自动补全提示支持的并不是太全面，为了更方便的写代码（其实是好多属性记不住）安装了这款插件，使用了一下感觉还是挺不错的。 

* For Windows 快捷键:  
* Jump to definition = ``Alt+Click``  
* Jump to definition = ``Control+Windows+Alt+Up``  
* Go back = ``Control+Windows+Alt+Left``  
* Manual CodeIntel = ``Control+Shift+space``

2.ColorPicker

ColorPicker - 一个内置的调色板选择插件. 调出面板的快捷键是：Ctrl+Shift+C

　　在写css的时候经常需要找颜色，这个插件正好满足，不多说上图：

3.FileDiffs

FileDiffs- 这个插件允许你看到SublimeText中两个不同文件的差异。你可以比较的对象可以是从剪贴板中复制的数据，或工程中的文件，当前打开的文件等。

　　偶尔会有比较文件差异的问题，这个插件还可以，不过对比后什么颜色具体代表什么经常记不住………………

4.DocBlockr

DocBlockr - 可以使你很方便地对代码建立文档。它会解析函数，变量，和参数，根据它们自动生成文档范式，你的工作就是去填充对应的说明。

　　使用法是在你定义的方法上输入/**然后再按回车即可。会自动生成变量和返回的相关信息

 /**
  * [sdf description]
  * @param  {[type]} id [description]
  * @param  {[type]} b  [description]
  * @param  {[type]} v  [description]
  * @return {[type]}    [description]
  */
function sdf(id,b,v){
    var obj=document.getElementById(id);
}

5.AdvancedNewFile

AdvancedNewFile- 一个快速创建指定目录下文件并命名的插件。安装完成后快捷键是：Ctrl + Alt + N

　　由于sublime自带的新建文件功能在保存的时候才会让你写名，有时候会感觉别捏，所以就选择了这个插件。
　　按过快捷键后会在底部出现新建文件框。默认在你打开的文件夹下，可以直接写目录然后写新建文件名。如果直接写/的话会建在所属盘符的根目录下。

6.jquery

　　可能会不定期的更新插件，因为是刚开始用所以暂时就先安装了这几个。后面随着不断的使用可能还会再安装其他插件。尽量在安装后就更新这篇文章。
